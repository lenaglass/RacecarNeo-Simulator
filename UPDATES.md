# RacecarNeo-Simulator Additions

## Controller

- PS5 Dualshock (haptics on USB; not supported on Bluetooth)
- Haptics: Speed (high rumble); Shock (low rumble)
- Lightbar: Gradient based on racer velocity.magnitude
- Right Stick: Chase cam controls (click toggles view)
- Triangle Button: Toggles IMU & Input HUD elements
- Square Button: Toggles Tron Mode

## Inverse Kinematics

- Triggers: Throttle (left:reverse / right:forward), exponential control curve
- Stearing: Speed adaptive sensitivity, exponential control curve
- Max speed: 1.0 (L1/R1 for ajustment down/up)

## Cameras

- "Fortnight" style lighting (bloom, auto-exposure adjustment)
- Camera shake on hard collisions

## Tron Mode

- Racer: Adds adaptive headlights and ground fx
- Shaders: Add a 'wireframe' overlay effect
- Adjustements to scene lighting & material emissivity
